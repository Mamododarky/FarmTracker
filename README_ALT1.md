# RS3 Farm Run Tracker v8

Host this folder on GitHub Pages and install `appconfig.json` in Alt1.

## Updating the existing repository
Replace the files in your local FarmTracker repository with this folder, commit, and push. GitHub Pages will redeploy automatically.

## Timer behavior
All deadlines are saved as absolute timestamps. Closing Alt1 does not erase them. The app cannot play sound while fully closed, but overdue patch and Manor Farm stage alarms are detected immediately when reopened. Browser/Alt1 audio may require one click per session.
